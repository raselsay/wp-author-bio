A WordPress author bio plugin that adds an extremeley flexible, custom about the author box below your posts for single and multiple authors.

== Description ==

WP Author Bio wedo is a WordPress author bio plugin that supports single and multiple authors with [Authors Plus](http://profiles.wordpress.org/raselself/). It offers tons of options to customize the box after WordPress posts about the author(s).

The author bio box is responsive and includes five social icon sets to choose from, with support for all the following social networks: Blogger, Facebook, Google+, LinkedIn,Twitter, WordPress, YouTube.

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Navigate to Author Bio -> Settings in order to customize any settings.
* Navigate to Users -> Your Profile and fill in the Contact Info, About Yourself & Author Signature Info fields. Do this for all authors on your website that have made posts (if you want this information to show up in their WP Author Bio wedo).

== Frequently Asked Questions ==

= How do I customize the look and feel of WP Author Bio wedo? =

* Once you've installed it simply go to your WordPress Admin Dashboard and then go to Settings > WP Author Bio wedo 

= How do I customize a user's wp Author Bio wedo? =

* Once you've installed it simply go to your WordPress Admin Dashboard and then edit the user for which you want to customize the WP Author Bio wedo. You'll see a section called "Author Signature Info" that contains WP Author Bio's wedo customization options for the user.
